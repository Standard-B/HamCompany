# 👷‍♀️求人&求職最大の人材市場 HamCompany👷‍♂️
### 人々が自身の才能と能力を幅広い仕事の世界で活用し、新たな挑戦を支援するシステム <br>
![메인페이지(로그인x)](https://github.com/Maengmo/HamCompany/assets/117720344/ada023b4-3757-4775-a9f0-585440977b6a)


<br>

## ⁉ プログラム紹介
  <p>人々が自分の才能や能力を広範な仕事の世界で活用し、新たな挑戦をサポートするプラットフォームを作ることに焦点を当てました。</p>
  <p>さまざまな業界や専門性を持つ人材が自分のプロフィールを登録し、それを通じて直接仕事を探し求める新しい形態の人材取引を探求しました。</p>
  <p>このアプローチは人材取引の透明性と効率性を向上させ、購入者にはワンクリックで人材を購入できる利便性を提供します。</p>
  <p>人々が自分の能力を最大限に活用し、より多くの機会を得ることを支援することが究極の目標です。</p>
  
  <br>
  
## ✔ 企画意図および主要業務
- ログイン, 会員登録
  - ログインの有効性検証、会員登録の有効性検証、Securityによって該当ユーザーに権限を付与します。<br><br>

- 人材を探す
  - 求職申し込みが完了した会員は求職市場に登録されます。事業者は選択した求職者から希望の求職者を選んで選択と支払いを行い、採用できます。支払いはカカオペイで行われます。<br><br>

- レビューボード
  - 求職経験者がレビューを投稿できるCRUD掲示板。<br><br>

- フリーボード
  - 事業者と求職者に分類されたCRUD掲示板で、自由に対話できます。<br><br>

- ポートフォリオ
  - 自身が進行したプロジェクトや仕事に関する情報を記録できる掲示板で、事業者はこれを通じてユーザーのポートフォリオを閲覧できます。<br><br>

- 求職者 マイページ
  - プロフィール、問い合わせ履歴、ポートフォリオ、応募履歴、レビューボード、フリーボードなど、CRUDが可能なマイページです。<br><br>
 
- 事業者 マイページ
  - 事業者プロフィール、事業者詳細プロフィール、ウィッシュリスト、レビュー、お問い合わせ履歴、注文履歴、フリーボードなど、CRUDが可能なマイページ
